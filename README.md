# Neurovascular Transfer Function Analysis

This repository provides a reproducible Python pipeline for dynamic cerebral autoregulation analysis using transfer function metrics derived from beat-to-beat blood pressure and cerebral blood flow velocity signals.

## Features

- Signal segmentation (START/END markers)
- Baseline normalization (% change)
- Welch power spectral density estimation
- Cross-spectral density
- Coherence, gain and phase computation
- Multi-panel publication-ready figures
- Group comparison framework (e.g., CT vs DP)

## Methods

Transfer Function Analysis (TFA) is implemented using SciPy's signal processing tools.
